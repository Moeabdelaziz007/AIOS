```
py65mon>=1.0.0      # 6502 monitor/debugger
```

## Integration Notes
- The BASIC-M6502 source code is in assembly language
- Python will provide the runtime environment and emulation
- py65 library handles 6502 microprocessor emulation
- Integration will create a Python wrapper around the assembly code
